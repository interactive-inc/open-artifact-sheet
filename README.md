# Open Artifact Sheet

https://developers.cloudflare.com/workers/framework-guides/web-apps/tanstack/

## Develop

以下のコマンドを実行します。

```
$ bun i
$ make apply-migration-local
$ bun run dev
```

## マイグレーションを実行する

以下のコマンドを実行します。

```
$ make create-migration
$ make apply-migration-local
```

## パッケージを更新する

以下のコマンドを実行します。

```
$ make update-packages
```
